# Travel-Box Bakend

이 프로젝트는 Express와 TypeScript를 사용하여 구현된 RESTful API입니다. MySQL, AWS S3, RDS, EC2, Redis를 사용하여 데이터를 관리하고 서비스를 제공합니다.

## setting

프로젝트를 실행하기 전에 다음 도구들이 설치되어 있어야 합니다:

- Node.js
- npm
- TypeScript
- MySQL
- AWS CLI (선택적)
